Project Name: Kovrr ShellHN

Description: The ShellHN has two capabilities
1: Display a list of the 40 most popular articles ordered by their rank
2: Upon selection of a given technology, predict the likelihood of the technology to appear in the following month in HN

How to install project:
Step 1: Make sure Python is installed on your computer
Step 2: Download the folder called "Kovrr" to a local repository

Running the project:
- In your terminal, navigate to the location where you have stored the directory called "Kovrr"
- The CLI takes in one argument which could be the number 1 or 2
- To display the top 40 articles on HN, write the command "python shellHN.py 1" in your terminal

- To predict the likelihood of a technology to appear in HN
Step 1: write the command "python shellHN.py 2" in your terminal
Step 2: A list of the technologies should appear.You will be prompted to enter the name of the technology you would like to make the prediction for.
